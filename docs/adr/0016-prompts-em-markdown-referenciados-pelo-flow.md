# Prompts em Markdown referenciados pelo flow

Prompts devem ser arquivos Markdown separados, em UTF-8, referenciados pelo fluxo de agente por identificador, caminho, versão e metadados. Isso facilita revisão, edição visual, comparação de mudanças e preservação correta de texto em português, sem embutir conteúdo longo no `agent.flow.json`.
